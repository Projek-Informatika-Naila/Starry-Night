We are from the Starry Code group, we present this website to visitors who open this website. Who do you think our members are? Mhmm... Check it out!

STAR CODE is an interactive educational game that we created inspired by the beauty and meaning of "Starry Night". We took this name from the combination of the words "Star" which symbolizes the inspiration of the night sky full of stars and "Code" as the basis of the programming world. We designed this program as a media for playing while learning for children and teenagers, so that they can hone their logic, creativity, and focus in a fun way.
